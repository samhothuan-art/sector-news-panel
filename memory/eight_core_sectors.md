## 选股原则（十大板块内）

### 优选标准
1. **板块龙头** - 市占率前三
2. **业绩兑现** - 订单/收入可见性高
3. **资金流入** - 主力净流入持续
4. **技术突破** - 均线多头排列或突破形态

### 回避标准
1. **业绩暴雷风险** - 预亏、减持
2. **高位放量滞涨** - 资金出货迹象
3. **流动性差** - 日均成交<5000万
4. **ST/退市风险**
