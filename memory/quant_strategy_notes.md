## 一、主流量化策略类型

### 1. 多因子选股策略
**核心思想**：通过多个指标筛选股票，构建投资组合

**常见因子：**
| 因子类型 | 具体指标 | 说明 |
|---------|---------|------|
| 价值因子 | PE、PB、PS、股息率 | 低估值选股 |
| 成长因子 | 营收增长率、净利润增长率 | 高成长选股 |
| 质量因子 | ROE、ROA、毛利率 | 优质公司 |
| 动量因子 | 近20日/60日涨幅 | 趋势跟踪 |
| 波动因子 | 换手率、振幅、标准差 | 低波动选股 |
| 资金流因子 | 主力净流入、大单占比 | 资金动向 |

**经典组合：**
- **小盘价值**：小市值 + 低PE + 高股息
- **质量成长**：高ROE + 营收高增长 + 合理PE
- **动量反转**：短期动量强劲 + 中期调整到位

---

### 2. 资金流向策略（适合尾盘）
**核心思想**：跟随主力资金动向，尾盘抢筹次日获利

**关键指标：**
- 主力净流入金额（大单+特大单）
- 主力净流入占比
- 量比（成交量/近期平均）
- 换手率（3-15%最佳）

**尾盘选股条件：**
```
1. 当日主力净流入 > 5000万
2. 涨幅 2-6%（不追高）
3. 换手率 3-15%（活跃但不极端）
4. 量比 > 1.5（放量）
5. 非ST、非688科创板
6. 流通市值 50-500亿（适中）
```

**风险控制：**
- 次日开盘冲高回落即止盈
- 设置-3%止损线
- 单票仓位不超过20%

---

### 3. 龙虎榜策略
**核心思想**：跟踪游资/机构席位动向

**数据来源：**
- Tushare `top_list` 接口（5000积分）
- 东方财富龙虎榜

**筛选条件：**
- 机构净买入 > 5000万
- 知名游资席位上榜
- 当日涨幅 < 7%（有空间）
- 非连续多日上榜（避免接盘）

---

### 4. 板块轮动策略
**核心思想**：资金流向哪个板块，就买哪个板块的强势股

**观察指标：**
- 板块主力资金净流入排名
- 板块涨幅排名
- 板块内涨停家数

**操作步骤：**
1. 14:40 看当日资金流入前10板块
2. 14:50 确认最终板块热度
3. 从热门板块中选最强个股
4. 次日早盘冲高卖出

---

### 5. 技术形态策略
**核心思想**：基于K线形态和指标信号交易

**常见形态：**
| 形态 | 说明 | 胜率 |
|------|------|------|
| 突破前高 | 放量突破近期高点 | 中等 |
| 缩量回踩 | 缩量回调至支撑位 | 较高 |
| 双底 | W底形态确认 | 较高 |
| 多头排列 | MA5>MA10>MA20>MA60 | 中等 |
| MACD金叉 | DIF上穿DEA | 中等 |

---

### 6. 事件驱动策略
**核心思想**：基于特定事件进行交易

**常见事件：**
- 业绩预增/预减
- 高送转
- 股权激励
- 重大合同
- 政策利好

**数据源：**
- Tushare `disclosure` 公告数据（1000元/年）
- 新闻舆情（1000元/年）

---

### 7. 筹码分布策略（高阶）
**核心思想**：分析持仓成本，判断支撑压力

**Tushare提供：**
- `cyq_chips` 每日筹码分布（10000积分）
- `cyq_perf` 筹码成本和胜率

**关键概念：**
- 获利盘比例 > 90% → 高风险
- 获利盘比例 < 10% → 潜在机会
- 筹码集中度 > 10% → 主力控盘

### 📊 推荐关注的思路

**adata项目（1nchaos）的核心理念：**
1. **多数据源融合** - 一个源挂了自动切换另一个（东财/同花顺/新浪）
2. **专注A股** - 不做大而全，只做深做透
3. **实时行情** - 用新浪/腾讯接口，免费且实时
4. **概念板块跟踪** - 同花顺+东财双数据源验证

**freevolunteer的实战经验：**
1. **可转债T+0** - 日内交易，不隔夜持仓
2. **自动打板** - 监测涨停前异动，自动发单
3. **WebSocket长连接** - 毫秒级行情推送

- 监测接近涨停的股票

- 次日高开即出
