# 尾盘抢筹策略 v2.0（修订版）

**制定时间**: 2026-02-27  
**核心逻辑**: 跟随主力资金流入板块，隔夜超短，分批建仓

---

## ✅ 策略优点
- 右侧交易，跟随主力
- 分批建仓，降低风险
- 高周转，资金利用率高
- 有明确止损纪律

---

## 🎯 选股条件（必须同时满足）

| 条件 | 标准 | 说明 |
|------|------|------|
| **大盘过滤** | 当天大盘跌幅 < 1.5% | 防系统性风险 |
| **板块资金** | 主力净流入 > 13亿 | 必须有资金关注 |
| **个股涨幅** | 2% < 涨幅 < 5% | 不是追高，也不是太弱 |
| **尾盘形态** | 尾盘30分钟不是直线拉升 | 防诱多 |

---

## 📊 建仓规则

### 第一步：尾盘建仓（14:50）
- 仓位：**30%**（不是50%）
- 目标：符合条件的板块滞涨股
- 价格：现价或挂低0.2%

### 第二步：第二天加仓（9:30-9:35）
- 条件：低开 **> 1.5%** 且 **量比 < 1.5**
- 仓位：**再加30%**
- 时间：9:35前必须决定

### 第三步：放弃场景
- 高开 > 3% → 不追，持有底仓观望
- 放量大跌（量比 > 2）→ 不加仓，直接止损
- 平开/小幅低开 → 观望，等10:30再决定

---

## 🚪 退出规则（严格执行）

| 情况 | 操作 | 优先级 |
|------|------|--------|
| **时间止损** | 10:30还没红，全走 | 高 |
| **纪律止损** | 跌破成本 -3%，全走 | 高 |
| **保本止盈** | 浮盈 > 2%，设保本单 | 中 |
| **目标止盈** | 达到目标价（+3%），分批走 | 中 |

---

## ⚠️ 风控红线

1. **单股仓位 ≤ 30%**（不是50%）
2. **同日最多操作 2 只股票**
3. **总仓位 ≤ 80%**（留20%现金）
4. **跌破-3%必须走**，不留幻想
5. **10:30不红必须走**，不浪费时间

## 💡 关键心法

1. **不追求完美买点** → 分批就是为了容错
2. **不扛单** → -3%是铁律
3. **不恋战** → 10:30不红就走，机会成本也是成本
4. **不贪婪** → +3%目标到了就走，不赚最后一个铜板
